# Dataset Overview

## Overview

This project uses the **Olist Brazilian E-Commerce Public Dataset**, a relational dataset containing transactional data from a Brazilian e-commerce marketplace.

The dataset captures the complete customer purchasing journey, including customers, orders, products, sellers, payments, reviews, and geographical information.

# Dataset Description

### Customers

Stores customer identification and location information.

---

### Orders

Stores order lifecycle information from purchase to delivery.

---

### Order Items

Stores every product purchased within an order.

---

### Payments

Contains payment methods, installment information and payment value.

---

### Reviews

Contains customer ratings and review comments after delivery.

---

### Products

Contains product category and physical characteristics.

---

### Sellers

Contains seller information and seller location.

---

### Geolocation

Contains latitude and longitude for Brazilian ZIP code prefixes.

---

### Category Translation

Maps Portuguese product category names to English.
